## Error Handling

<Expandable title="Common Error Codes" default-open="false">
  Learn about standard HTTP status codes and error responses.

  ### 4xx Client Errors
  
  - `400 Bad Request` - Invalid request parameters
  - `401 Unauthorized` - Missing or invalid authentication
  - `403 Forbidden` - Insufficient permissions
  - `404 Not Found` - Resource does not exist
  
  ### 5xx Server Errors
  
  - `500 Internal Server Error` - Unexpected server error
  - `502 Bad Gateway` - Upstream service unavailable
  - `503 Service Unavailable` - Service temporarily down
</Expandable>

<CodeGroup tabs="Python,JavaScript">
  ```python
import requests

def create_document(api_key, title, content):
    url = "https://api.aaron.ai/v1/documents"
    headers = {
        "Authorization": f"Bearer {api_key}",
        "Content-Type": "application/json"
    }
    data = {
        "title": title,
        "content": content,
        "format": "markdown"
    }
    
    response = requests.post(url, headers=headers, json=data)
    
    if response.status_code == 201:
        return response.json()
    else:
        raise Exception(f"API Error: {response.status_code} - {response.text}")
```
  ```javascript
async function createDocument(apiKey, title, content) {
    const response = await fetch('https://api.aaron.ai/v1/documents', {
        method: 'POST',
        headers: {
            'Authorization': `Bearer ${apiKey}`,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({
            title,
            content,
            format: 'markdown'
        })
    });
    
    if (response.ok) {
        return await response.json();
    } else {
        throw new Error(`API Error: ${response.status} - ${await response.text()}`);
    }
}
```
