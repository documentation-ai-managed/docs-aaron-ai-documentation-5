## API Configuration

Configure API endpoints and webhooks to extend Aaron AI's functionality.

<CodeGroup tabs="JSON,YAML">
  ```json
{
  "webhooks": [
    {
      "url": "https://your-app.com/webhook",
      "events": ["document.updated", "document.created"],
      "secret": "your-webhook-secret"
    }
  ],
  "api": {
    "rate_limit": 1000,
    "timeout": 30
  }
}
  ```
  ```yaml
webhooks:
  - url: https://your-app.com/webhook
    events:
      - document.updated
      - document.created
    secret: your-webhook-secret
api:
  rate_limit: 1000
  timeout: 30
  ```
</CodeGroup>

## Security Settings

Configure security policies to protect your documentation and ensure compliance.

| Setting | Description | Default Value |
|---------|-------------|---------------|
| Password Policy | Minimum password requirements | 8+ characters |
| Session Timeout | Automatic logout after inactivity | 30 minutes |
| Two-Factor Auth | Require 2FA for all users | Disabled |
